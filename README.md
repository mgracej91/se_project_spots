# Project 6 Spots

An image sharing platform.

## Description

This project is a social media platform that can be used for posting images. Spots allows the user to upload their photo with name and description, create new posts giving other users the option to like the posts, and also has the option for deleting posts.

## Tech stack

- HTML
- CSS
- JavaScript
- Form Validation
- Responsive design
- Asynchronous operations
- BEM file structure
- Git

## Features

This webpage has many features for users to interact with such as setting their custom profile with name, photo, and a description! This webpage also allows image posting that can be added and/or removed and other users have the ability to like these images. There are many form validations and handlers to ensure the forms are user friendly. My webpage is also responsive to many devices including mobile view.
